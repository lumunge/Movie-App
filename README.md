### REACT MOVIE APP

### ABOUT

A movie searching application

### TECHNOLOGIES USED:

    * Reactjs
    * Material UI
    * react-share

### FEATURES

    * Search for movies
    * Add movie to favorites
    * Remove Movie from favorites
    * Share on social media (facebook)
